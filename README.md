# Resume-Matcher Pro
# AI Resume Screening System
A machine learning-based web app that analyzes resumes and matches them with job descriptions using NLP.
- ## Use Case
Helps recruiters filter candidates faster and helps job seekers improve resumes.
## Features
- Resume parsing (PDF)
- Keyword extraction
- Similarity scoring using TF-IDF
- Job description matching
- ## Tech Stack
- Python
- Streamlit
- Scikit-learn
- NLTK
- PyPDF2
- ## Live Demo
(https://resume-matcher-3298.streamlit.app/)
